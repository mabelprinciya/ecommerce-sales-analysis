# 📊 E-Commerce Sales Analysis Using Python

## 📌 Project Overview
This project analyzes e-commerce sales data using Python. The dataset contains information such as Order ID, Date, Product, Category, Quantity, and Price. The main goal is to extract meaningful insights from sales data.

## 📁 Dataset
- File Type: CSV (ecommerce_sales.csv)
- Columns: Order_ID, Date, Product, Category, Quantity, Price

## 🛠️ Technologies Used
- Python
- Google Colab

## 📚 Libraries Used
- Pandas – Data handling and analysis
- NumPy – Numerical operations
- Matplotlib – Data visualization

## 📊 Analysis Performed
- Revenue calculation (Quantity × Price)
- Top selling products analysis
- Monthly sales trend analysis
- Category-wise sales distribution

## 📈 Visualizations
- Bar Chart (Top Products)
- Line Chart (Monthly Sales)
- Pie Chart (Category Sales)

## 🎯 Key Insights
- Electronics category generated highest revenue
- March recorded highest sales
- Top products contributed major revenue share

## 🚀 How to Run
1. Open Google Colab
2. Upload dataset (CSV file)
3. Run Python code cells step by step
4. View results and visualizations
